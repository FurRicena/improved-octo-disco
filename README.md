# RICENAFU – Restaurant Management System

> A lightweight restaurant management system built with Spring Boot and Vue.

## ✨ Features
- **User Management** – registration, login, role-based access
- **Menu Management** – CRUD for dishes, category filtering
- **Order Management** – order creation, status updates, export functionality

## 🛠 Tech Stack
- **Backend**: Java 21, Spring Boot 3.x, Mysql
- **Frontend**: TypeScript, Vue 3, Vite, Element Plus, Axios, Pinia
- **Tools**: Maven, Git, Apifox
